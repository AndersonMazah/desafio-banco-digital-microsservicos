# Desafio Banco Digital

Este repositório deverá conter um código de um desafio técnico de um sistema bancário baseado em microsserviços.

## Ele deve ter:

- Cadastro e autenticação de usuários
- Gestão de contas correntes
- Movimentações financeiras (crédito e débito)
- Transferências entre contas da mesma instituição
- Consulta de saldo


## Tecnologias que deverão ser utilizadas

- .NET 8
- Dapper
- Mediator
- Swagger
- Kafka
- KafkaFlow
- SQLite
- Docker
- Docker Compose
- xUnit
